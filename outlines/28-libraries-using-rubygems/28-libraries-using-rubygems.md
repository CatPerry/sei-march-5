## Using Ruby Gems

```bash
$ gem install mygem
$ gem uninstall mygem
$ gem list --local
```

## Bundler

_Bundler allows us to group a set of gems together in a package that can be distributed with our apps for another developer to use_

```bash
$ gem install bundler
$ touch Gemfile
$ bundle
```
